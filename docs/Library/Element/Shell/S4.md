# S4

Four Node Shell Element

The `S4` shell element uses `Minlin` plate element combined with `GQ12` element.

* Number of Nodes: 4
* Number of DoFs: 6

## Syntax

```
element S4 (1) (2...5) (6) (7)
# (1) int, unique element tag
# (2...5) int, node i, j, k, l
# (6) int, material tag
# (7) double, thickness
```
